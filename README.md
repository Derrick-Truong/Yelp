# Welcome to Ice Cream Delight, a Yelp Clone project!

Explore and review various ice cream parlors with this website. Create an account, log in, and enjoy the features like favorites, searching, and integration with Google Maps.

## Demo Link

You can access the live demo of the Ice Cream Delight Yelp Clone project [here](https://example.com/ice-cream-delight-demo).

## Features

### Account Creation and Login

- Create an account and securely log in.
- Explore the site using a guest or demo login.

### Restaurants

- Post, edit, and delete ice cream parlor listings.
- View a feed of recently reviewed ice cream parlors.

### Reviews

- Post, edit, and delete reviews on ice cream parlors.
- Read reviews posted by other users.

### Favorites

- Save your favorite ice cream parlors.
- Remove favorites from ice cream parlors.
- See the number of users who have favorited a parlor.
- View a page showing your favorited parlors.

### Google Maps Integration

- Display the location of an ice cream parlor using Google Maps.

### Search Functionality

- Search for specific ice cream parlors using keywords.

## Technologies Used

The Ice Cream Delight Yelp Clone project utilizes the following technologies:

- React.js: JavaScript library for building user interfaces.
- Node.js: JavaScript runtime environment.
- Express.js: Web application framework for Node.js.
- js-cookie: Library for handling cookies in JavaScript.

Feel free to explore the Ice Cream Delight Yelp Clone project and enjoy reviewing and discovering new ice cream parlors!
